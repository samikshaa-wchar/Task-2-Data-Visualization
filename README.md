
# Task 2 - Data Visualization and Storytelling

## Objective
Create visualizations that convey business insights using charts and graphs.

## Tools Used
- Power BI
- Canva
- GitHub

## Dataset
Superstore Sales Dataset

## Dashboard Features
- KPI Cards
- Sales by Category
- Profit by Region
- Segment Wise Sales
- Business Insights Panel

## Key Insights
- Technology category generated highest sales
- West region achieved maximum profit
- Consumer segment contributed most sales

## Outcome
Learned dashboard design, data visualization and storytelling.
